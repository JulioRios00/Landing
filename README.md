1. Clone the repository:

```bash
git clone https://github.com/JulioRios00/Landing.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Visit http://localhost:3000 to view your portfolio!
